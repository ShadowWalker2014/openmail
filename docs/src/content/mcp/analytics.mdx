# Analytics Tools

2 tools for pulling email performance data.

## get_analytics

Get workspace-wide analytics for the last 30 days, including contacts, sends, open rate, click rate, and unsubscribes.

No parameters required.

> "What is my overall email performance over the last 30 days?"

**Returns:**

```json
{
  "contacts": 12453,
  "sends": 58920,
  "opens": 14317,
  "clicks": 3002,
  "unsubscribes": 118,
  "openRate": 24.3,
  "clickRate": 5.1,
  "period": "30d"
}
```

Note: `openRate` and `clickRate` are **percentages** (e.g. `24.3` = 24.3%), not fractions.

**Useful questions to ask:**

- "How many contacts do I have?"
- "What's my open rate over the last 30 days?"
- "How many emails have I sent recently?"

## get_broadcast_analytics

Get detailed analytics for a specific broadcast.

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|:--------:|-------------|
| `broadcastId` | string | ✓ | Broadcast ID (`brd_xxx`) |

> "How did my March Newsletter perform? Give me open rates, click rates, and unsubscribes."

```
Tool: get_broadcast_analytics
{ "broadcastId": "brd_abc123def456" }
```

**Returns:**

```json
{
  "broadcastId": "brd_abc123def456",
  "sentCount": 4812,
  "openCount": 1169,
  "clickCount": 245,
  "openRate": 24.3,
  "clickRate": 5.1,
  "open": 1169,
  "click": 245,
  "unsubscribe": 12,
  "bounce": 9
}
```

Note: `openRate` and `clickRate` are **percentages** (e.g. `24.3` = 24.3%).

**Tip:** Combine with `list_broadcasts` to find a broadcast by name:

> "Find my most recent broadcast and show me its performance metrics."

The agent will call `list_broadcasts` to find the ID, then `get_broadcast_analytics` to retrieve metrics.
